# Bangla Signboard Dataset

This dataset containsBangla signboard images for detection, text recognition, and classification.

## Download

You can download the dataset here:  
[Download Bangla Signboard Dataset](https://drive.google.com/file/d/1e5CP7MbIoW-gUjPeac7-8vIk4su9Vbjb/view?usp=drive_link)

## Usage

- Extract the ZIP file after downloading.  
- Use the images for training, testing, or benchmarking signboard detection and recognition models.
